# Scripts PowerShell
Os scripts em PowerShell foram desenvolvidos para simular o uso das aplicações, eles executam comandos do Android Debug Bridge (ADB) para interações de gestos com a tela do dispositivo, a fim de padronizar a velocidade das interações tanto na aplicação em Flutter como na aplicação em React Native. Desta forma os ruídos que a interação manual poderia ocasionar são evitados.
